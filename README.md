# Virtusa_Core_Java_Project-Smart-Pay
Smart pay utility biller is a core java console application which calculates utility bills with slab rates and applies tax .
## Oops concepts used 
- Encapsulation
- Abstraction
- Polymorphism 
## And also added basic input validations such as 
- Negative readings
- Previous readings > Current readings
- Empty uase names
